# v0.9.3 UI Theme Patch

목표: 현재 기능 테스트용 UI를 원본 반츄 4.1.4에 가까운 형태로 조정한다.

## 변경 사항
- 키보드 높이 축소
- 버튼 간격 축소
- 배경색을 원본형 회색으로 변경
- 키 색상/테두리/모서리 반경 조정
- 버튼 그림자 및 상단 하이라이트 추가
- 글자 크기 축소
- 보조문자 위치/색상 조정
- Space 키를 텍스트 대신 원본형 아이콘으로 그림
- DEL 키를 원본형 텍스트+아이콘 조합으로 그림

## 테스트 기준
- 한글 자판이 원본보다 과하게 커 보이지 않아야 한다.
- 키 간격이 원본처럼 촘촘해야 한다.
- DEL/Space가 원본과 비슷한 시각적 느낌을 가져야 한다.
- 빠른 입력 성능 패치(v0.9.2)는 유지되어야 한다.
