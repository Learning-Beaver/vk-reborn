# v0.9.2 Turbo Patch

## Goal
Improve key response speed and reduce missed input during fast typing.

## Changes
- Process normal key input on `ACTION_DOWN` instead of `ACTION_UP`.
- Reduce long-press threshold from 360ms to 280ms.
- Separate long-press from normal input.
- Long-press now replaces the immediately inserted normal key with the long-press output.
- Keep partial invalidation and cached key rectangles from v0.9.1.

## Tests
- Fast Hangul typing: `안녕하세요안녕하세요안녕하세요`
- Fast English typing: `qwertyuiopasdfghjkl`
- Fast number typing: `1234567890`
- Long-press Hangul key: normal jamo should be replaced by number.
- Long-press English/Symbol key: normal char should be replaced by sub label.
- Long-press DEL: continuous deletion.
