# v0.9.6 Composite Vowel Patch

## 확정 규칙

```text
ㅘ = 6+3
ㅙ = 6+3+9
ㅚ = 6+9
ㅝ = 66+3
ㅞ = 66+3+9
ㅟ = 66+9
ㅢ = 999
```

## 테스트 케이스

```text
0+66+3     -> 워
0+66+3+9   -> 웨
0+66+9     -> 위
1+6+3      -> 과
1+6+3+9    -> 괘
1+6+9      -> 괴
9+9+9      -> ㅢ
```

## 구현 메모

기존 엔진은 `66+3`을 `ㅜ+ㅏ`로 해석한 뒤 결합하지 못해 `우ㅏ`로 출력했다.
이번 버전에서 `VowelComposeTable`에 아래 규칙을 추가했다.

```text
ㅜ + ㅏ -> ㅝ
ㅝ + ㅣ -> ㅞ
ㅜ + ㅣ -> ㅟ
ㅗ + ㅏ -> ㅘ
ㅘ + ㅣ -> ㅙ
ㅗ + ㅣ -> ㅚ
ㅡ + ㅣ -> ㅢ
```
