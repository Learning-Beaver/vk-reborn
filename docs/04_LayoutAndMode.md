# VK Reborn v0.4 - Layout / Mode

## 한글 모드

- 좌측 상단 `▦ / ▼`: 키보드 내리기
- `[123]`: 숫자 자판 전환
- `#`: 직전 자모 변환
- `[ABC]`: 영문 자판 전환
- 각 한글 키의 우측 상단 작은 숫자/기호: 롱프레스 입력

```text
[▦▼] [ㄱㅋ¹] [ㄴㅁ²] [ㅏㅓ³] [DEL]
[123] [ㄷㅌ⁴] [ㄹ⁵]  [ㅗㅜ⁶] [Space]
[#]   [ㅂㅍ⁷] [ㅅ⁸]  [ㅣㅡ⁹] [ENTER]
[ABC] [ㅈㅊ?] [ㅇㅎ⁰] [한~]  [.]
```

## 숫자 모드

`[123]`에서 진입한다.

```text
[▦▼] [1] [2] [3] [DEL]
[SYM] [4] [5] [6] [Space]
[ABC] [7] [8] [9] [ENTER]
[한글] [,] [0] [-] [.]
```

## 현재 구현 범위

- 한글/숫자/영문/특수문자 모드 전환 골격
- 롱프레스 이벤트 처리
- `#` 변환 이벤트 처리
- 원본 반츄식 회색 버튼 기반 커스텀 View 초안
