# v0.8 English Keyboard

## Scope
- Restore original Vanchu English QWERTY layout.
- Implement English Shift state machine.
- Add long-press sub-label output for English keys.

## Shift State Machine

| Current | Shift press | Next |
|---|---|---|
| OFF | 1st | ONCE |
| ONCE | 2nd | LOCK |
| LOCK | 3rd | OFF |

## Typing behavior

| State | Input |
|---|---|
| OFF | lowercase |
| ONCE | uppercase one character, then OFF |
| LOCK | uppercase continuously |

## UI behavior
- OFF: normal lowercase labels.
- ONCE: uppercase labels.
- LOCK: uppercase labels and blue dot on Shift key.

## Tests
- ABC mode → `a` = `a`
- Shift + `a` = `A`, then next `b` = `b`
- Shift + Shift + `a` + `b` = `AB`
- Shift + Shift + Shift + `a` = `a`
- Long press `q` = `1`
- Long press `a` = `~`
- Long press `s` = `!`
