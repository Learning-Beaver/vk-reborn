# v0.9.4 UI Height Restore

## Goal
Restore the keyboard total height while keeping the improved v0.9.3 visual style.

## Changes
- `onMeasure()` height restored from `width * 0.38` to `width * 0.58`.
- Minimum height restored from `280` to `300`.
- Top/bottom padding set to `8dp`.
- Key gap remains compact at `3dp`.
- Label position moved slightly upward.
- Space icon position moved slightly upward.

## Test
- Compare total keyboard height with v0.9.2.
- Confirm v0.9.3 key style is preserved.
